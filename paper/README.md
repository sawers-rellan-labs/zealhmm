# `paper/` — manuscript pointer

The manuscript itself does **not** live here. Per the GEMMA lab standard, the
preprint is drafted in the **RILAB LaTeX template**
(<https://github.com/RILAB/rilab_template>), then uploaded to Overleaf and synced
to a dedicated paper repo. `zealhmm` produces the **figures, tables, and rendered
analysis notes**; the LaTeX lives separately.

  near-isogenic lines across sequencing modalities (nilHMM)."*
- **Central novelty:** ancestry-caller parameters can be calibrated from cheap
  `simcross` simulations instead of scarce high-coverage truth samples.
