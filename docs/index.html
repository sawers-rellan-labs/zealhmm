zealhmm

Simulation-calibrated HMM ancestry calling for near-isogenic lines across sequencing modalities — analysis repo for the nilHMM methods paper (Oxford Bioinformatics).

GitHub repo · nilHMM package

Thesis. Ancestry-caller parameters (rigidity r, error err, emission means) can be calibrated from cheap simcross simulations rather than from scarce high-coverage truth samples, and transfer across the Skim (low-cov WGS) and BrB (3′ RNA-seq) modalities.
